# برنامه لیست کارها (To-Do List)

این یک برنامه ساده و مینیمال برای مدیریت کارها است که با HTML/JS خالص نوشته شده و به دیتابیس Supabase متصل می‌شود.

## راهنمای راه‌اندازی

### ۱. تنظیم دیتابیس (Supabase)
قبل از اینکه برنامه کار کند، باید جدول مربوطه را در دیتابیس خود بسازید:
1. فایل `setup_database.sql` را باز کنید و محتویات آن را کپی کنید.
2. به پنل کاربری خود در [Supabase](https://supabase.com) بروید.
3. از منوی سمت چپ، وارد بخش **SQL Editor** شوید.
4. روی **New Query** کلیک کنید، کد را پیست کرده و دکمه **Run** را بزنید.

### ۲. آپلود در گیت‌هاب (GitHub)
برای اینکه سایت شما آنلاین شود:
1. یک مخزن (Repository) جدید در گیت‌هاب بسازید.
2. فایل‌های `index.html` و `README.md` را در آن آپلود کنید (یا از طریق ترمینال push کنید).

### ۳. فعال‌سازی GitHub Pages
1. وارد صفحه مخزن خود در گیت‌هاب شوید.
2. به بخش **Settings** بروید.
3. از منوی سمت چپ، گزینه **Pages** را انتخاب کنید.
4. در بخش **Build and deployment** و زیرشاخه **Branch**، گزینه `main` (یا `master`) را انتخاب کرده و دکمه **Save** را بزنید.
5. پس از چند دقیقه، لینک سایت شما در بالای همان صفحه ظاهر می‌شود (مثلاً: `https://your-username.github.io/repo-name/`).

حالا می‌توانید از برنامه خود استفاده کنید!
